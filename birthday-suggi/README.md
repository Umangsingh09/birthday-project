# Birthday Website for Suggi

A romantic birthday website created with love for Suggi.

## Features
- Cinematic intro with animated text
- Photo galleries of memories and journey
- Time counter since first meeting
- Music player with romantic songs
- 20 reasons why I love you list
- Responsive design with glassmorphism effects
- Interactive animations and effects

## Music Files Required

Please add the following music files to the `music/` folder for the music player to work:

1. `perfect.mp3` - Perfect by Ed Sheeran
2. `until-i-found-you.mp3` - Until I Found You by Stephen Sanchez
3. `yellow.mp3` - Yellow by Coldplay

You can download these songs legally from your music library or streaming service.

## How to Run

1. Open a terminal in the project directory
2. Run: `python -m http.server 8000`
3. Open your browser and go to `http://localhost:8000`
